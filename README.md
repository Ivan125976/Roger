Quecto Roger 2 и выше на архитектуре MLP - .NET 9.0)

Инструкция к Quecto Roger 2.0 / Yocto Roger 2.1
1. Установить .NET 9.0
2. Запустить программу Quecto Roger v.2.0.exe / Yocto Roger v.2.1.exe
3. Отредактировать файл knowledge.know или создать свой файл знаний в формате "x y z, " где X - первое значение, Y - второе, а Z - ожидаемый вывод. По умолчанию стоит .know файл сгенерированный как складывание 2 чисел
4. Настройки: LR (Learning Rate) - чем меньше тем качественее будет обучаться нейросеть. Passes - чем больше тем лучше усвоит информацию нейросеть. Knowledge файл - файл знаний на котором будет обучаться нейросеть. В Yocto Roger 2.1.exe есть параметр DropOut - это функция защищающая нейросеть от переобучения, советуется ставить или 5% или 10%, но с DropOut требуется более маленький LR и больше Passes
