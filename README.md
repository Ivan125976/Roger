Данная нейросеть написана на .NET (Quecto Roger 1 и ниже по архитектуре LP — .NET 8.0, Quecto Roger 2 и выше по архитектуре MLP — .NET 9.0)

Инструкция к Quecto Roger 2.0 / Yocto Roger 2.1

1. Установить .NET 9.0

2. Запустить программу Quecto Roger v.2.0.exe / Yocto Roger v.2.1.exe

3. Отредактировать файл знаний.знать или создать свой файл знаний в формате "xyz", где X - первое значение, Y - второе, а Z - ожидаемый вывод. По умолчанию стоит .know файл сгенерирован как сложение 2 чисел

4. Настройки: LR (Learning Rate) - чем меньше тем качественее будет обучаться нейросеть. Проходит - чем больше тем лучше усвоит информацию нейросеть. Файл знаний - файл знаний, на котором будет обучаться нейросеть. В Yocto Roger 2.1.exe есть параметр DropOut - это функциональная нейросеть от переобучения, советуется или ставится 5% или 10%, но с DropOut требуется более маленький LR и больше проходов.
